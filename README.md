# c-web-server

## __MUST HAVE__
Linux/macOS: 
- *make*
- *GCC*

Windows (never tested on windows):
- *some sort of **make** port for windows*
- *a C compiler*


## Running the server 

```
git clone <insert-link-of-repo>
cd ./c-web-server
make build
./main
```
Then open your browser and type in the link `localhost:42069`,
or whatever port you set it to.
